# H2 node-todo
A simple node.js todo list (create, remove, list) using fs &amp; yargs.


# H3 instructions
node app.js --help

> create a todo item

node app.js create -item="Take out the trash"

> remove an existing item

node app.js remove -item="name of the item you want removed"

> list all todo items

node app.js list
